# Changelog

## 1.1.0 — 2026-07-16

- Consolidated specialist UI workflows into one primary `perfect-ui-skill`.
- Added focused review, codebase audit, audit-to-plan, opportunity discovery, plan execution, and reconciliation modes.
- Added strict motion craft guidance covering purpose/frequency gates, easing, duration, origin, physicality, interruptibility, direct manipulation, velocity handoff, momentum projection, rubber-banding, springs, performance, and reduced-motion alternatives.
- Added a compact motion vocabulary so agents can name and specify effects precisely.
- Expanded typography guidance with optical sizing, size-specific tracking/leading, and text-scaling rules.
- Expanded component contracts for press feedback, dialogs/sheets, tooltips, gestures, collection stability, and motion states.
- Expanded async feedback and dependency-isolation guidance.
- Upgraded the static audit with motion, reduced-motion, hover-gating, duration, layout-animation, and `will-change` checks plus category filtering and severity summaries.
- Added motion and UI-plan bootstrap templates.
- Added package tests and GitHub Actions validation.
- Added MIT license and third-party attribution for adapted ideas from `emilkowalski/skills`.

## 1.0.0 — 2026-07-16

- Initial portable Agent Skill release.
- Added strict UI operating workflow and source-of-truth rules.
- Added foundations, responsive, component-state, accessibility, motion/performance, dashboard, migration, quality-gate, stack-adapter, and anti-pattern references.
