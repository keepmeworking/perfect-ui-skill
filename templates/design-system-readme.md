## Responsive strategy
Define compact, medium and expanded behavior for navigation, content, tables, charts, dialogs and forms.

## Component policy
Reuse before creating. Every shared component requires complete states, accessibility, responsive, theme and content-extreme contracts. Recurring visual values must become tokens.

## Accessibility target
WCAG 2.2 AA for web and equivalent native guidance.

## Definition of done
Use the `perfect-ui-skill` quality gates and maintain the state matrix and audit evidence.
