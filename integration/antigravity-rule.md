# Mandatory Perfect UI rule

Whenever a task affects a user interface, invoke `perfect-ui-skill` and follow it as the governing UI workflow. This includes screens, pages, components, CSS/theme code, responsive behavior, accessibility, forms, tables, charts, navigation, motion, loading/error states, and visual audits. Preserve the existing brand/design system, use semantic tokens and canonical components, implement all relevant states, and verify the rendered UI before completion.
